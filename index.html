ℹ️ 关于本站

🔧 工作原理
本站数据由 GitHub Actions 在每天凌晨 2:00（北京时间）自动调用七鱼开放平台 API 抓取，生成静态 JSON 文件后通过 Cloudflare Pages 提供服务。GitHub 仓库保持私有。

📅 数据更新

每日凌晨 2:05（北京时间）自动抓取前一天和当天的统计数据。手动触发更新请前往 GitHub 仓库的 Actions 页面。

💬 会话分析

批量会话导出需要企业开通七鱼「会话导出」API 权限。如需查看具体会话内容，请在七鱼管理后台查看。

📊 数据范围

本站展示的数据总览、坐席绩效、质量报告均来源于七鱼开放平台统计类 API。数据延迟约 1 天。

🔐 安全性

API 密钥存储在 GitHub Secrets 中，不会暴露在公开代码中。数据文件为公开只读。
