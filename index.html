ℹ️ 关于本站

🔧 工作原理
本页面仅展示会话记录（由 GitHub Actions 每日凌晨缓存）。
实时统计数据（总览/坐席绩效/质量报告）请本地运行后端服务： http://localhost:5890

💬 会话数据

每日凌晨 2:00（北京时间）自动导出并缓存前一天会话数据。需开通七鱼「会话导出」API 权限。

📊 实时统计

数据总览、坐席绩效、质量报告为实时数据，需本地运行 Flask 后端（python backend/app.py）访问 localhost:5890。

🔄 更新频率

会话数据：每日凌晨缓存一天。实时统计：打开 localhost:5890 即刻刷新。

🔐 安全性

API 密钥存储在 GitHub Secrets 中，不会暴露在公开代码中。数据文件为公开只读。
